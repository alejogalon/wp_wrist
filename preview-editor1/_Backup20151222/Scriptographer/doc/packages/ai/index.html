The Adobe Illustrator Package


The Adobe Illustrator functionality. All prototypes for working with documents, graphic objects, vector paths, text objects, fonts, raster images, filters, etc.
It also contains a description of objects and functions present in the Global Scope, and basic prototypes such as Point, Rectangle, Color and Matrix.
