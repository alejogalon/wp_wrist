ItemContainer

ItemContainer acts as container of ADM Items that share a layout and can be added to a parent container, e.g. another ItemContainer or a Dialog. This is part of the AWT Layout wrapper code. ItemContainer itself wrapps a AWTItemContainer that does the actuall layout work.
